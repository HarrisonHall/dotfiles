# Bootstrap

This configuration can be extended to additional systems. Utilize the minimal
`mise.toml` (copy to `~/.config/mise/config.toml`) after reviewing some of the
bootstrap dependencies.

## Bootstrap

Requirements are necessary to get the rest of the configuration running.

### Requirements

- make
- bash
- fish
- git
- tree
- mosh
- wget
- zip/unzip

### Tier 2

- colordiff
- file
- gnupg
- kbd
- xdg-utils
- macchina
- nitch
- yt-dlp
- systemctl-tui
- procs
- fselect
- ouch
